Get Started with Retriever

1. Setup Base Infrastructure

First, let's set up the base infrastructure for Retriever. This will provision the necessary AWS resources to run Retriever.

2. Prepare the CLI

Next, configure the Retriever CLI to connect to your infrastructure and prepare for deployment.

3. Provide Your AWS Info and Deploy Retriever!

Finally, provide your AWS credentials and deploy Retriever to your infrastructure. Once deployed, you'll be able to use our custom MCP server to query and analyze your telemetry data. Or you can check out our Jaeger, Prometheus, and AlertManager dashboards to see your data in more detail!
