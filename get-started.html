Get Started with Retriever

Follow these steps to spin up Retriever and start monitoring your applications with end-to-end observability.

1. Prerequisites

Before you begin, ensure you have the following installed:

  • Docker and Docker Compose (or your preferred container runtime)
  • Node.js (version 18 or higher)
  • Git

2. Clone the Repository

Clone the Retriever repository from GitHub:

git clone https://github.com/TeamRetriever/retriever.git
cd retriever

3. Install Dependencies

Install the required dependencies:

npm install

4. Configure Environment Variables

Create a .env file in the root directory and configure the necessary environment variables:

cp .env.example .env
# Edit .env with your configuration

5. Start Retriever

Start the Retriever platform using Docker Compose:

docker-compose up -d

Or if you prefer to run it locally:

npm start

6. Access the Dashboard

Once Retriever is running, access the dashboard at:

http://localhost:3000

7. Connect Your Application

Follow the integration guide to connect your application to Retriever and start collecting traces.

Need Help?
